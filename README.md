## Installation

Clone the repo:
```
 git clone https://github.com/sndrjhlncgr/Youtube-Video-and-MP3-Downloader.git
```
 ### Install
 
 Install the node modules:
 
 ```bash
  $ cd react-youtube-downloader
  $ npm install
 ```
 ### Run the project

 Start the project:
 
 ```
  $ npm start
 ```
  build the project:
  ```
  $ npm run build
 ```
 ***Note***: in the future maybe i will remove `npm run build`.
